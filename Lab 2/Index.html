1. Heading Element:

    • Syntax:

      • <h1>This is heading 1</h1>

      <h2>This is heading 2</h2>

      <h3>This is heading 3</h3>

    • Output:

      • This is heading 1

      This is heading 2

      This is heading 3

  2. Paragraph Element

    • Syntax:

      • <p> this is a paragraph.
      this is an example of
      Paragraph. </p>

    • Output:

      • this is a paragraph. this is an example of Paragraph.

  3. Line break Element

    • Syntax:

      • <p>a<br>p<br>p<br>l<br>e<br>s </p>

    • Output:

      • a
      p
      p
      l
      e
      s

  4. Blockquote Element

    • Syntax:

      • <blockquote>"This is a blockquote
      This is an example of
      blockquote"</blockquote>

    • Output:
      • "This is a blockquote This is an example of blockquote"

  5. Phrase Element

    • Syntax:
      • <b>bold</b> text.
      <em>enphasized</em> text.
      <i>italicized</i> text.
      <mark>mark</mark> text.
      <small>small</small> text.
      <strong>strong</strong> text.
      <sub>sub</sub> text.
      <sup>sup</sup> text.
    • Output:
      • bold text. enphasized text. italicized text. mark text. small text. strong text. sub text. sup text.

  6. Lists

    1. Lists Unordered

      • Syntax:

        • <li>example a</li>
        <li>example b</li>

      • Output:
      • example a
      • example b

    2. Lists Ordered

      • Syntax:

        • <ol>
            <li>example a</li>
            <li>example b</li>
        </ol>

      • Output:
        1. example a
        2. example b

    3. Description list

      • Syntax:
        • <dl>
            <dt>Triangle</dt>
                <dd>Pyramid</dd>
            <dt>Square</dt>
                <dd>Cube</dd>
        </dl>
      • Output:
        •     
        Triangle
                
        Pyramid
            
        Square
                
        Cube

  7. Special Characters

    • Syntax:
      • "copyright": &copy
      <br> "copyright": &lt
      <br> "copyright": =&gt
      <br> "copyright": =&amp
    • Output:
      • "copyright": ©
      "copyright": <
      "copyright": =>
      "copyright": =&

  8. Anchor Element

    1. Absolute link
      • Syntax:
        • Link to a different website: <a href="http://google.ca">www.google.ca </a>
      • Output:
        • Link to a different website: www.google.ca
    2. Relative link
      • Syntax:
        • Link to a different page on the same website: <a href="../Lab1/index.html">Lab1 </a>
      • Output:
        • Link to a different page on the same website: Lab1
    3. E-mail hyperlink
      • Syntax:
        • Link to mail program connected with browser or a messages is displayed <a href="mailto:me@gmail.com">me@gmail.com </a>
      • Output:
        • Link to mail program connected with browser or a messages is displayed me@gmail.com

  9. Meter Element

    • Syntax:
      • <meter value="100" min="0" max="100"> max 100 </meter>
      <meter value="50" min="0" max="100"> max 50 </meter>
      <meter value="10" min="0" max="100"> max 0 </meter>
    • Output:
      • max 100 max 50 max 0

  10. Progress Element

    • Syntax:
      • <progress value="700" max="1000"> 700</progress>
    • Output:
      • 700

  11. Datalist Control

    • Syntax:
      • <label for="fruit">Fruit</label>
      <input type="text" name="fruit" id="fruit" list="fruits">
      <datalist id="fruits">
          <option value="Peaches">
          <option value="Apples">
          <option value="Pears">
      </datalist>
    • Output:

    • Syntax:
      • <label for="Hunger"> Express how hungry you are between 1 and 10 <br>
      Low <input type="range" name="myChoice" id="Hunger" min="1" max="10"> High
      </label>
    • Output:

  13. Calendar Control

    • Syntax:
      • <label for="theDate">What day?</label>
      <input type="date" name="theDate" id="theDate">
